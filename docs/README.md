## 📝 文档编号说明

文档采用 `类别编号-文档编号-中文标题.md` 的扁平化命名格式:

- **00**: 元文档(管理规范)
- **01**: 项目概述
- **02**: 架构设计
- **03**: 技术实现
- **04**: 开发指南
- **05**: 设计规范
- **06**: API 参考
- **07**: 部署运维
- **08**: 测试文档
- **09**: 附录资源

所有文档直接放在 `docs` 目录下,便于维护和查找。
